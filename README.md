# Finance_Transaction
This project focuses on analyzing financial transaction data using SQL to uncover insights around transaction patterns, balances, and potential inconsistencies.

The goal was to simulate a real-world scenario where raw financial data needs to be cleaned, structured, and queried to support decision-making.

What I Worked On
Data cleaning and preparation
Writing SQL queries for transaction analysis
Aggregating inflows and outflows
Identifying trends and unusual patterns
Improving query structure and readability
Tools Used
SQL (core analysis)
GitHub (version control and documentation)
What I Learned

This project helped me strengthen my SQL fundamentals, especially around joins, aggregations, and filtering. It also highlighted the importance of writing efficient and readable queries, not just ones that “work.”
